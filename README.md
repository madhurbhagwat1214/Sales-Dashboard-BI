📊 Sales Analytics Dashboard | Power BI
📌 Project Overview

This project presents an interactive Sales Analytics Dashboard built using Power BI and Excel. The dashboard provides insights into sales performance, product profitability, and customer retention, helping businesses make data-driven decisions.

The project is divided into three analytical pages:

Executive Sales Overview
Product Profitability Analysis
Customer Insights & Retention Analysis
🎯 Objectives
Analyze overall sales and profitability.
Identify top-performing products and regions.
Evaluate product-level profit contribution.
Understand customer purchasing behavior.
Measure customer retention and segment performance.

🚀 Project Outcome

The dashboard transforms raw sales data into actionable business insights by combining KPI tracking, profitability analysis, customer segmentation, and retention metrics. It enables stakeholders to monitor performance, identify growth opportunities, and support strategic decision-making.
